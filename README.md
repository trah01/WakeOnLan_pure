# 使用说明

IP地址的最后一段建议设为255，如电脑ip为192.168.1.28，则设置为192.168.1.255，这样貌似是对整个局域网进行广播，当然你也可以设为你的IP，如果能生效的话

MAC地址使用‘-’‘:’‘.’这3个符号进行分隔即可，当然，不能混搭

长按中间按钮即可进行修改
